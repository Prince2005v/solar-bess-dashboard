# Solar & Battery Energy Storage System Dashboard

## Project Overview

This project analyzes the performance of a decentralized Solar and Battery Energy Storage System (BESS) implemented across facilities in Kenya.

The dashboard was created using Tableau Public and focuses on operational efficiency, financial savings, and environmental sustainability metrics.

## KPIs Included

* Energy Generated (kWh)
* Diesel Savings
* Battery Utilization (%)
* CO₂ Emissions Avoided (kg)

## Tools Used

* Tableau Public
* CSV Dataset
* Data Visualization
* KPI Analysis

## Dashboard Features

* Interactive KPI dashboard
* Trend analysis
* Battery performance monitoring
* Environmental impact analysis

## Files Included

* Tableau Workbook (.twbx)
* Dataset (.csv)
* Dashboard Screenshot
